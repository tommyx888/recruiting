📧 Gmail API Setup - Kompletný Návod

Čas potrebný: 15-20 minút | Náklady: Zadarmo (100 emailov/deň)

💡 Tip: Tento návod vás prevedie cez celý proces nastavenia Gmail API pre váš recruiting systém.

🔧 Krok 1: Google Cloud Console

1.1 Vytvorte projekt:

  1. Choďte na Google Cloud Console
  2. Kliknite na dropdown s názvom projektu (hore vľavo)
  3. Kliknite "New Project"
  4. Zadajte názov: Recruiting Email System
  5. Kliknite "Create"

1.2 Povoľte Gmail API:

  1. V ľavom menu choďte do "APIs & Services" > "Library"
  2. Vyhľadajte "Gmail API"
  3. Kliknite na "Gmail API"
  4. Kliknite "Enable"
✅ Hotovo! Gmail API je teraz povolené pre váš projekt.

🔑 Krok 2: Vytvorte Service Account

2.1 Choďte do Credentials:

  1. V ľavom menu choďte do "APIs & Services" > "Credentials"
  2. Kliknite "Create Credentials" > "Service Account"

2.2 Vyplňte údaje:

Service account name: recruiting-email-service Service account ID: recruiting-email-service Description: Service account for sending recruiting emails

Kliknite "Create and Continue"

2.3 Pridajte role (voliteľné):

Môžete preskočiť alebo pridať "Editor" rolu. Kliknite "Continue"

2.4 Vytvorte kľúč:

  1. Kliknite na vytvorený Service Account
  2. Choďte do "Keys" tab
  3. Kliknite "Add Key" > "Create new key"
  4. Vyberte "JSON"
  5. Kliknite "Create"
  6. Stiahnite JSON súbor
⚠️ Dôležité: Uchovájte JSON súbor na bezpečnom mieste. Obsahuje citlivé údaje!

📧 Krok 3: Gmail Setup

3.1 Vyberte Gmail účet:

Použite Gmail účet, ktorý chcete použiť na posielanie emailov (napr. noreply@iacgroup.com)

3.2 Povoľte 2-Factor Authentication:

  1. Choďte do Google Account Security
  2. V "Signing in to Google" kliknite "2-Step Verification"
  3. Postupujte podľa inštrukcií na zapnutie 2FA

3.3 Vytvorte App Password:

  1. V "Signing in to Google" kliknite "App passwords"
  2. Vyberte "Mail" a "Other (Custom name)"
  3. Zadajte názov: "Recruiting System"
  4. Kliknite "Generate"
  5. Skopírujte 16-znakový kód (napr. abcd efgh ijkl mnop)
✅ Hotovo! Máte App Password pre Gmail API.
