📧 Mailgun Setup - Rýchly Návod

Čas potrebný: 5-10 minút | Náklady: Zadarmo (10,000 emailov/mesiac)

💡 Výhody Mailgun: Rýchle nastavenie, môžete posielať na akékoľvek adresy, 10,000 emailov/mesiac zadarmo.

🔑 Krok 1: Registrácia na Mailgun

1.1 Zaregistrujte sa:

  1. Choďte na mailgun.com
  2. Kliknite "Sign Up"
  3. Vyplňte registračný formulár
  4. Potvrďte email

1.2 Vytvorte API Key:

  1. Po prihlásení choďte do "API Keys"
  2. Kliknite "Create API Key"
  3. Pomenujte kľúč: "Recruiting System"
  4. Skopírujte API Key (začína sa s "key-")
✅ Hotovo! Máte Mailgun API Key

🌐 Krok 2: Overenie domény

2.1 Pridajte doménu:

  1. Choďte do "Domains" v Mailgun dashboard
  2. Kliknite "Add New Domain"
  3. Zadajte: mg.iacgroup.com (alebo vašu doménu)
  4. Vyberte "EU" region (pre GDPR compliance)
  5. Kliknite "Add Domain"

2.2 Pridajte DNS záznamy:

Mailgun vám poskytne DNS záznamy, ktoré musíte pridať do vašej domény:

# TXT Record pre verifikáciu Name: mg.iacgroup.com Value: v=spf1 include:mailgun.org ~all # CNAME Record pre tracking Name: email.mg.iacgroup.com Value: mailgun.org
⚠️ Dôležité: Pridajte DNS záznamy do vašej domény a počkajte na overenie (5-30 minút).
