📧 SendGrid - Rýchle nastavenie

Čas potrebný: 3-5 minút | Náklady: Zadarmo (100 emailov/deň)

💡 Výhody SendGrid: Žiadne nastavovanie domény, môžete posielať na akékoľvek adresy, 100 emailov/deň zadarmo.

🔑 Krok 1: Registrácia na SendGrid

1.1 Zaregistrujte sa:

  1. Choďte na sendgrid.com
  2. Kliknite "Start for Free"
  3. Vyplňte registračný formulár
  4. Potvrďte email

1.2 Vytvorte API Key:

  1. Po prihlásení choďte do "Settings" > "API Keys"
  2. Kliknite "Create API Key"
  3. Vyberte "Restricted Access"
  4. Povoľte "Mail Send" permissions
  5. Pomenujte kľúč: "Recruiting System"
  6. Skopírujte API Key (začína sa s "SG.")
✅ Hotovo! Máte SendGrid API Key

📧 Krok 2: Overenie Sender Identity

2.1 Single Sender Verification (Najrýchlejšie):

  1. Choďte do "Settings" > "Sender Authentication"
  2. Kliknite "Verify a Single Sender"
  3. Vyplňte formulár:
    • From Name: Recruiting System
    • From Email: noreply@iacgroup.com
    • Reply To: noreply@iacgroup.com
    • Company Address: Vaša adresa
  4. Kliknite "Create"
  5. Skontrolujte email a kliknite na verifikačný link
✅ Hotovo! Sender identity overená - môžete posielať emaily!
